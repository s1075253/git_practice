# git_practice
本專案是用於 Git 和 GitHub 的自我練習，適合剛開始學習版本控制的學生和開發者。透過本專案，你將熟悉如何操作 Git 的基本命令，並學習如何使用 GitHub 進行協作式開發。

## 目的
學習並掌握 Git 和 GitHub 的基本操作，包括複製儲存庫、進行變更、提交修改以及將變更推送到遠端儲存庫。這些都是日常軟體開發中的基礎技能。

## 要求
- 安裝好 Git。
- 有一個 GitHub 帳號。

# Try it
1. Fork 這個專案
2. Clone你所fork的專案到你本地的環境
3. 添加新檔案和寫一些內容
4. Commit 你的內容
5. Push儲存庫到github上
6. 練習Markdown語法
